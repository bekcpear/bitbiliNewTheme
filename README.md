## bitbiliNewTheme

一个适用于 Pelican 所生成静态网站的双色主题。

其是我博客的主题，访问 [我的博客](https://bitbili.net) 以查看 Demo

* 暗色/亮色模式
* 离线阅读
* 简约风格
* noscript 友好
* 等
